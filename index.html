Closing Time

HTML5 | CSS | Javascript | Google API

This website enables users to figure out what places are open nearby. The idea came from many experiences of being out at night friends and trying to figure out what places nearby are still open. It utilizes the HTML5 geolocation API and Google Maps/Places API to determine where you are and what places nearby are open. Right now, I'm working on a mobile version that I hope to get in the App Store.
